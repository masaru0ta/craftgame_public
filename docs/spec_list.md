# 機能一覧

ブラウザで動作するボクセルゲーム（マインクラフトクローン）の機能一覧。
GAS上でプレイ可能な形でデプロイする。

|  #   | カテゴリ       | 機能                         | 概要              |
|------|--------------|------------------------------|------------------------|
| 1-1  | データシート  | データシート設計と確認        | スプレッドシートの設計と状態確認                                            |
| 1-2  | ツール        | GAS API                      | スプレッドシートのデータを読み書きする API                                   |
| 1-3  | ツール        | 標準ブロックエディタ          | 標準ブロックの読込・メッシュ生成・編集UI・保存                               |
| 1-4  | ツール        | カスタムブロックエディタ       | カスタムブロックの読込・メッシュ生成・編集UI・保存                           |
| 1-5  | ツール        | カスタムブロック／当たり判定   | カスタムブロックの当たり判定編集と簡易チェック                    |
| 1-6  | ツール        | ブロック形状一覧の表示と編集   | ブロック形状一覧から、標準ブロック・カスタムブロックの編集ができる |
| 1-7  | ツール        | データファイル一覧画面       | 利用するスプレッドシートを切り換える  |
| 1-8  | ツール        | マルチブロックエディタ         | 複数まとめて設置するアイテムの編集 |
| 2-1  | ゲーム        | １チャンク生成・表示テスト     | チャンクデータ管理、メッシュ生成、テスト描画 |
| 2-2  | ゲーム        | チャンク管理テスト             | 複数チャンクの動的生成・保存・読込のテスト |
| 2-3  | ゲーム        | LoD設定と表示テスト            | 距離に応じた描画詳細度の切り替え |

# フェーズ

| # | フェーズ名 | 説明 | 必要な機能 |
|---|-----------|------|-----------|
| 1 | スプレッドシートの設計 | 通常ブロック、カスタムブロックをどう格納するか考える | 1-1 |
| 2 | 標準ブロックの編集 | 標準ブロックを読込・編集・保存できるようにする | 1-2, 1-3 |
| 3 | カスタムブロックの編集 | カスタムブロックを読込・編集・保存できるようにする。当たり判定も編集。 | 1-4, 1-5 |
| 4 | ブロック管理ツール | ここまでの機能を利用し、ブロック形状一覧を表示し、標準ブロック・カスタムブロックの編集が出来るツール | 1-6 |
| 5 | データファイル管理 | 複数のスプレッドシートを切り替えて利用できるようにする | 1-7 |
| 6 | ゲーム基盤 | チャンクシステム、メッシュ生成、テスト描画 | 2-1 |
| 7 | チャンク管理 | 複数チャンクの動的生成・保存・読込 | 2-2 |
| 8 | LoD システム | 距離に応じた描画詳細度の切り替え | 2-3 |
