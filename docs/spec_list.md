# 機能一覧

ブラウザで動作するボクセルゲーム（マインクラフトクローン）の機能一覧。
GAS上でプレイ可能な形でデプロイする。

|  #   | カテゴリ       | 機能                      | 概要                                                             |
|------|--------------|----------------------------|------------------------------------------------------------------|
| 1-1  | データシート  | データシート設計と確認       | スプレッドシートの設計と状態確認                                 |
| 1-2  | ツール        | GAS API                   | スプレッドシートのデータを読み書きするAPI                         |
| 1-3  | ツール        | 標準ブロックエディタ        | 標準ブロックの読込・メッシュ生成・編集UI・保存                    |
| 1-4  | ツール        | カスタムブロックエディタ     | カスタムブロックの読込・メッシュ生成・編集UI・保存                |
| 1-5  | ツール        | カスタムブロック／当たり判定 | カスタムブロックの当たり判定編集と簡易チェック                    |
| 1-6  | ツール        | ブロック形状一覧の表示と編集 | ブロック形状一覧から標準ブロック・カスタムブロックの編集ができる  |
| 1-7  | ツール        | データファイル一覧画面       | 利用するスプレッドシートを切り換える                              |
| 1-8  | ツール        | マルチブロックエディタ       | 複数まとめて設置するアイテムの編集                                |
| 2-1  | ゲームコア     | １チャンク表示              | １チャンクのデータを表示するテスト。テクスチャアトラス化   |
| 2-2  | ゲームコア     | 連続チャンク生成・保存・読込 | チャンクの動的生成・保存・読み込みを連続して行うテスト |
| 2-3  | ゲームコア     | LoD設定と表示テスト |  |
| 2-4  | ゲームコア     | 移動テスト | プレイヤーが1人称視点で移動、物理演算（重力、ジャンプ、AABB衝突判定）するテスト |
