# 機能一覧

ブラウザで動作するボクセルゲーム（マインクラフトクローン）の機能一覧。
GAS上でプレイ可能な形でデプロイする。

|  #  | カテゴリ     | 機能                   | 概要 |
|-----|-------------|---------------------------|----------------------------------------------|
| 1-1 | データシート | データシート設計と確認      | スプレッドシートの設計と状態確認                 |
| 1-2 | ツール       | GAS API                  | スプレッドシートのデータを読み書きする API         |
| 1-3 | ツール       | 標準ブロックエディタ       | 標準ブロックの読込・メッシュ生成・編集UI・保存     |
| 1-4 | ツール       | カスタムブロックエディタ    | カスタムブロックの読込・メッシュ生成・編集UI・保存 |
| 1-5 | ツール       | カスタムブロック/当たり判定 | カスタムブロックの当たり判定編集と、簡易チェック   |
| 1-6 | ツール       | ブロック一覧の表示と編集    |  |
| 1-7 | ツール       | データファイル一覧画面      |  |
| 1-8 | ツール       | テクスチャ一覧画面          |  |
| 1-9 | ツール       | マルチブロックエディタ      |  |


# フェーズ

| # | フェーズ名 | 説明 | 必要な機能 |
|---|-----------|------|-----------|
| 1 | スプレッドシートの設計 | 通常ブロック、カスタムブロックをどう格納するか考える | 1-1 |
| 2 | 標準ブロックの編集 | 標準ブロックを読込・編集・保存できるようにする | 1-2, 1-3 |
| 3 | カスタムブロックの編集 | カスタムブロックを読込・編集・保存できるようにする | 1-4 |
